## 🐾 About

**Pawfect Beauty** is a luxury dog grooming salon in Las Vegas, Nevada, founded by **Mihsha** and **Jenny** — two professional groomers with a combined **32 years of experience**. Every dog is treated like family, in a high-end boutique environment designed for comfort, care, and style.

## 🛠️ Tech Stack

This is a static, hand-coded website built with vanilla web technologies — no frameworks, no dependencies, just clean and fast HTML, CSS, and JavaScript.

| Layer | Technology |
|---|---|
| Markup | HTML5 |
| Styling | CSS3 |
| Interactivity | Vanilla JavaScript |
| Hosting | GitHub Pages |
| Domain | Custom — `pawfectbeauty.com` |
| Booking | [groomer.io](https://groomer.io) integration |
| SEO | `sitemap.xml`, `robots.txt`, `llms.txt`, Google Search Console |

## 🚀 Deployment

The site is deployed via **GitHub Pages** with a custom domain.

1. Push to the `main` branch
2. GitHub Pages auto-deploys from the repo root
3. The `CNAME` file routes traffic from `pawfectbeauty.com`

No build step required — the site is fully static.
